# Portfolio Dung Vo

Portfolio personnel statique de Dung Vo, pret a publier sur GitHub Pages.

## Arborescence A Uploader Sur GitHub

Quand tu uploades manuellement sur GitHub, ouvre le dossier `projet1` et selectionne directement ces fichiers/dossiers. Ne selectionne pas le dossier `projet1` lui-meme.

```txt
projet1/
|-- index.html
|-- style.css
|-- script.js
|-- README.md
|-- .gitignore
|-- .nojekyll
|-- assets/
|   |-- DUNGVO_CV.pdf
|   |-- photocv.png
|   |-- aminus.png
|   |-- health-hackathon.png
|   `-- wardrobe-wizard.png
`-- docs/
    `-- cahier_des_charges_site_personnel_codex.md
```

Important : `index.html` doit etre directement a la racine du depot GitHub, au meme niveau que `style.css`, `script.js` et `assets/`.

## Publier Manuellement Sur GitHub

1. Cree un depot sur GitHub, par exemple `portfolio-dung`.
2. Dans le depot vide, clique sur `uploading an existing file`.
3. Ouvre ton dossier local :

```txt
C:\Users\33769\summerproject\projet1
```

4. Selectionne tout le contenu du dossier, pas le dossier lui-meme :

```txt
index.html
style.css
script.js
README.md
.gitignore
.nojekyll
assets/
docs/
```

5. Upload puis clique sur `Commit changes`.

6. Va dans :

```txt
Settings -> Pages
```

7. Choisis :

```txt
Source: Deploy from a branch
Branch: main
Folder: /root
```

8. Attends quelques minutes. Le site sera disponible ici :

```txt
https://dung-kathy.github.io/portfolio-dung/
```

Si ton depot s'appelle `Dung-Kathy.github.io`, l'adresse sera :

```txt
https://dung-kathy.github.io/
```
